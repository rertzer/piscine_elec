# Hardware

- ATmega328p
- AHT20 temperature sensor
- 7-segment display
- I2C 16-bit I/O Expander
- SPI RGB LEDs

# Main Topics

- AVR registers
- Timers and duty cycle
- Interrupt Service Routine (ISR)
- Analog to Digital Converter (ADC)
- EEPROM
- UART
- I2C
- SPI
